# サイドメモ（Side Memo）

Chrome拡張機能のサイドパネルでメモを作成・管理できるシンプルなアプリです。

## 機能

- メモの一覧表示
- 新規メモの作成
- 既存メモの編集
- メモの削除

## 技術スタック

- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## 開発方法

### 必要条件

- Node.js (v16以上)
- npm

### インストール

```bash
# 依存関係のインストール
npm install
```

### 開発サーバー起動

```bash
# 開発サーバーの起動
npm run dev
```

### ビルド

```bash
# 本番用ビルド
npm run build
```

ビルドしたファイルは `dist` ディレクトリに出力されます。

## インストール方法（Chrome拡張機能として）

1. `npm run build` を実行してビルドする
2. Chromeで `chrome://extensions` を開く
3. 「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリックする
5. ビルドした `dist` ディレクトリを選択する

インストール後、ツールバーに表示されるアイコンをクリックするとサイドパネルが開きます。
